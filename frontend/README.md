# TestHub Frontend

Frontend application for TestHub educational platform built with React and Vite.

## Features

- ✅ Authentication (Login/Register)
- ✅ Exam Platform with multiple choice questions
- ✅ Consultation System with weekly schedules and chat
- ✅ Video Platform
- ✅ PDF Notes Sharing
- ✅ Student Discussion Forum
- ✅ Field Information Pages
- ✅ User Profile Management

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will run on `http://localhost:3000`

## Backend Connection

Make sure the Django backend is running on `http://localhost:8000`

The API base URL is configured in `src/services/api.js`

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
